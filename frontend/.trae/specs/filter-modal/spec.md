# 看板筛选器配置弹窗 - 产品需求文档

## Overview
- **Summary**: 实现一个筛选器配置弹窗，允许用户添加、配置和管理看板的筛选字段，支持多种筛选类型和配置选项。
- **Purpose**: 解决看板数据筛选的需求，使用户能够根据不同维度对看板数据进行过滤，提高数据可视化的灵活性和准确性。
- **Target Users**: 看板编辑人员、数据分析师、业务用户。

## Goals
- 实现筛选器配置弹窗，包含添加筛选字段功能
- 支持数据集选择、字段选择、筛选器类型配置
- 支持筛选默认值设置和生效图表选择
- 实现保存和取消操作，确保配置数据的正确性

## Non-Goals (Out of Scope)
- 不支持复杂的筛选逻辑组合（如AND/OR条件）
- 不支持自定义筛选器样式
- 不涉及后端API的实现，仅关注前端交互

## Background & Context
- 当前系统中筛选器功能处于开发中状态，点击筛选器按钮仅显示提示信息
- 系统使用Ant Design组件库，需要保持UI风格一致
- 看板编辑页面已存在，需要在此基础上添加筛选器配置功能

## Functional Requirements
- **FR-1**: 筛选器按钮点击事件处理，打开配置弹窗
- **FR-2**: 筛选字段管理功能，支持添加、选择和配置筛选字段
- **FR-3**: 数据集选择功能，支持选择关联的数据集
- **FR-4**: 数据集字段选择功能，根据所选数据集动态加载字段
- **FR-5**: 筛选器类型选择功能，支持多选、单选、日期区间三种类型
- **FR-6**: 筛选默认值配置功能，根据筛选器类型显示不同的配置组件
- **FR-7**: 生效图表选择功能，支持选择筛选规则的生效范围
- **FR-8**: 保存和取消操作，支持配置数据的提交和放弃

## Non-Functional Requirements
- **NFR-1**: 响应式设计，确保在不同屏幕尺寸下正常显示
- **NFR-2**: 交互流畅，操作反馈及时
- **NFR-3**: 表单验证，确保配置数据的完整性和合法性
- **NFR-4**: 错误处理，提供清晰的错误提示信息

## Constraints
- **Technical**: 使用React、Ant Design组件库
- **Business**: 最多支持添加10个筛选字段
- **Dependencies**: 依赖后端API提供数据集和图表信息

## Assumptions
- 后端API已提供获取数据集和图表列表的接口
- 筛选器配置数据将通过后端API进行保存和加载

## Acceptance Criteria

### AC-1: 筛选器按钮点击打开配置弹窗
- **Given**: 用户在看板编辑页面
- **When**: 点击「筛选器」按钮
- **Then**: 弹出筛选器配置弹窗，显示配置界面
- **Verification**: `human-judgment`

### AC-2: 添加筛选字段功能
- **Given**: 筛选器配置弹窗已打开
- **When**: 点击「+添加筛选字段」按钮
- **Then**: 左侧字段列表中新增一个未命名筛选字段，并自动选中
- **Verification**: `human-judgment`

### AC-3: 数据集选择功能
- **Given**: 已选择一个筛选字段
- **When**: 在「数据集」下拉框中选择一个数据集
- **Then**: 「数据集字段」下拉框自动加载该数据集下的所有字段
- **Verification**: `human-judgment`

### AC-4: 筛选器类型选择功能
- **Given**: 已选择数据集和字段
- **When**: 选择不同的筛选器类型（多选、单选、日期区间）
- **Then**: 筛选默认值配置组件同步变更
- **Verification**: `human-judgment`

### AC-5: 筛选默认值配置功能
- **Given**: 已选择筛选器类型
- **When**: 配置筛选默认值
- **Then**: 系统保存默认值配置
- **Verification**: `human-judgment`

### AC-6: 生效图表选择功能
- **Given**: 已配置筛选字段
- **When**: 在「生效图表」下拉框中选择图表
- **Then**: 系统记录筛选规则的生效范围
- **Verification**: `human-judgment`

### AC-7: 保存操作
- **Given**: 已完成筛选器配置
- **When**: 点击「确定」按钮
- **Then**: 系统校验配置合法性，通过后保存配置并关闭弹窗
- **Verification**: `human-judgment`

### AC-8: 取消操作
- **Given**: 已修改筛选器配置
- **When**: 点击「取消」按钮
- **Then**: 弹出确认弹窗，确认后放弃修改并关闭配置弹窗
- **Verification**: `human-judgment`
